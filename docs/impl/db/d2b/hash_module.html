Hash module

hash:sha256

Generates a SHA256 of a given string.

Parameters

  • data: a string to hash

Return value String: hexadecimal hash

Author

  • @SuperFola

Example

(import "hash.arkm")
(print (hash:sha256 "arkscript")) # c56d0bb03e9e03e7000f7b596cdb333c315523af7e4781f02610c690441c43dd
