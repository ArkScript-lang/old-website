switch

(let switch (fun (_value _tests) (...)))
Takes a value to match against a list of [possible values, function to run if it matched]
Note : Once the value is matched, it stops and doesn't try any other values.

Parameters

  • _value: value to match
  • _test: list for test value, composed by sub lists who contained possible value and function to run if matched

Author

@SuperFola

Example


(switch 12 [
    [1 (fun () (print "the value is one"))]
    [12 '(print "quoted code blocks also works")]
    [12 '{ (let b "ok") (print b ", quoted begin blocks work as well") }]
])
