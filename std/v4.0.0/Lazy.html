lazy:eval

(let lazy:eval (fun (f) (...)))
Take a function, and return a closure to lazy evaluate it
Note :

Parameter

  • f: the function to lazy evaluated

Author

@SuperFola

Example


(let complex_stuff (fun () {
    # do complex work in the function
    42 }))
(let lazy (lazy:eval complex_stuff))
(print (lazy))
