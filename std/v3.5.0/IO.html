Builtin (print values)

Note : No separator is put between the values. Adds a \n at the end

Parameter

  • values: the values to print

io:writeFile

Builtin (io:writeFile filename mode content)
Write content to a file, given an optional mode (default: "w"). Return nil
Note :

Parameters

  • filename: path to the file to open
  • mode: (optional), either "a" (append) or "w" (write)
  • content: can be any valid ArkScript value

Author

@SuperFola

Example


(io:writeFile "hello.json" "{\"key\": 12}")
(io:writeFile "truc.txt" "a" 12)
